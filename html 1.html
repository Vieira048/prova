O que é HTML?

HTML é a sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a base de praticamente todas as páginas da internet.
Pensa assim: o HTML é como o “esqueleto” de um site. Ele organiza o conteúdo textos, imagens, links, botões para que o navegador (como Chrome ou Safari) consiga mostrar tudo na tela.

  • Organiza e estrutura o conteúdo

  • Diz ao navegador o que mostrar e como interpretar

  • Permite a navegação entre páginas

Qual o papel dele na Web?

O HTML serve para organizar e mostrar o conteúdo das páginas na Web. Ele diz ao navegador o que é texto, título, imagem e link, funcionando como a estrutura básica de um site.
